## Учебный проект MDS-25

**Технологии проекта:**
- Streamlit
- FastAPI
- SQLite
- Machine Learning

**Запуск:**
* > uvicorn backend.api:app --reload
* > streamlit run frontend/Home.py
 
Проект построен на основе учебного проекта, созданного на семинарах, но исходного кода почти не осталось, использовался в основном только скелет/структура проекта.
В будущем планируется расширить функционал и добавить предсказание оттока для большого количества клиентов через CSV-файл. 
Проект выполнен на англйиском языке с целью сохранения возможности использовать его в портфолио в международной среде.
